1. ```npx create-react-app [폴더명] --typescript```
2. ```npm install --save @types/react @types/react-dom``` // 타입 정의
3. ```npm install --save-dev typescript``` //타입스크립트 패키지 설치
4. ```npx typescript --init``` //타입스크립트 설정파일 생성
